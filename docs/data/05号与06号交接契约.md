# 05号与06号交接契约

## 给 05 号业务语义负责人

从 Real 业务库读取 `metrics(metric_id, metric_name, metric_definition, metric_unit, value_scale)`、`institutions(institution_id, institution_name)` 与 `derived_dimensions(dimension_name, definition)`。`value_scale` 表示展示值为 `metric_value_scaled / 10 ^ value_scale`。05 号仅审核并定义衍生公式与题库口径，不修改 Real 数据库。

## 给 06 号测试负责人

评测程序先读取 `data/private/official/active_release.json`，再使用其中相同 `run_id` 对应的独立评测库。必须调用 `validate_active_release()` 核对业务库与评测库的 `run_id`、源 SHA-256 和发布路径；不一致时停止评测。题目文本、标准答案、逐题结果只保存在 Git 忽略的私有目录。
