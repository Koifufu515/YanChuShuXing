# 官方数据审计报告

本报告仅记录聚合结果。导入程序自动审计表头、空字段、公式单元格、主键、事实组合唯一性、机构与指标关联、指标名称一致性、日期格式和有限数值。

当前受控工作簿包含 13 家机构、21 项指标、10 项衍生维度、132,678 条指标事实和 200 条评测题目。机构编号、指标编号、问题编号唯一；事实组合键无重复；事实表机构与指标关联通过；日期均可标准化。

| 工作表 | 检查 | 当前结果 | 公开证据 |
|---|---|---|---|
| 机构信息表 | 主键、空值、重复 | 通过 | 行数与规则 |
| 指标清单表 | 主键、单位、名称 | 通过 | 行数与规则 |
| 衍生维度说明 | 名称、口径非空 | 通过 | 行数与规则 |
| 指标数据表 | 组合键、外键、日期、数值、名称一致性 | 通过 | 行数、日期范围与规则 |
| 问题答案清单 | 主键、字段非空 | 通过 | 行数与规则 |
