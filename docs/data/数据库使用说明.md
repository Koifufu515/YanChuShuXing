# Real 数据库使用说明

1. 将官方工作簿放入 `data/private/official/`，确认该路径被 Git 忽略。
2. 调用 `scripts.data.import_official_workbook.import_workbook()`，传入工作簿、`data/real` 和 `data/private` 路径。
3. 使用 `scripts.data.validate_real_database.validate_active_release()` 校验 active manifest 和六类查询。
4. 后端接入前由项目负责人审核 Real Schema 与数据库路径配置。

业务查询仅读取 active manifest 指向的 `bankinsight_real.db`。评测程序读取独立 `questions.db`；两者不得交叉使用。
